Quizr
=====

A quiz application. 

This is only a skeleton for app that students have to made during postgraduate
studies "Programowanie aplikacji internetowych" on Wyższa Szkoła Nauk
Humanistycznych i Dziennikarstwa (Poznań, Poland).
